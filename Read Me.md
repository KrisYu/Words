### 保存

文件可以保存，保存的后缀是 wnb ( Word Notebook)


### 打开保存的文件

⚠️ 请将保存的文件 drop 到 App上打开

⚠️ 或者使用App的 File 下的 Open 打开


不知道为什么，在我的机器上的几个账户测试，间歇性双击并不能加载出文件内容，但是如果使用 drop 和 File 下的 Open 一定会成功
